Frequently Asked Questions

1. What is Aperture User Management System?

Aperture is a secure and powerful platform for managing users, roles, and access controls within any organization. It helps you maintain structure, reduce administrative overhead, and enhance security.

2. How do I add a new user?

You can easily add a new user by navigating to the Add User page, filling out the form with required details such as name, email, role, and clicking submit.

3. Can I assign or change user roles?

Yes. Visit the Manage User section, click on the user, and select the desired role from the dropdown menu. Save your changes to apply them.

4. Is my data secure?

Absolutely. We use bank-level encryption, role-based access controls, and security audits to ensure your data is always protected.

5. Can I export user data?

Yes. You can export user data in CSV or Excel formats from the dashboard for reporting and backup purposes.

6. What browsers are supported?

Aperture supports all modern browsers including Chrome, Firefox, Safari, and Microsoft Edge.

7. Is there a mobile version?

Yes, our platform is fully responsive and can be accessed on phones and tablets for on-the-go management.
